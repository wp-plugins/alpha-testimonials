== Description ==

**Alpha Testimonials** is Responsive Testimonials Showcase Plugin that comes in two layouts: Default and alternative. It adapts to every screen and device. It comes packed with widget that you can include in any sidebar.

Alpha Testimonials [Demo](http://themeflection.com/plugins/wordpress/alpha)

For support visit my [support forum](http://themeflection.com/support/)

Main Features:

*   Responsive
*   2 layouts
*   Modern Look
*   Easy to Customize
*   Simple integration via copy-paste Shortcode
*   Packed with widget


== Installation ==

Alpha Testimonials can be installed in two different ways:

*  Via WordpPress uploader
*  or via FTP Client

1- Steps to follow to install plugin via WordPress uploader

*   go to WordPress admin panel and click on "plugins" from the dashboard menu.
*   Then from Plugins -> Add New -> Upload -> Choose file -> and then choose alpha-testimonials.zip and press Open-> Install Now -> Activate Plugin

2- Steps to follow to install plugin via FTP

*   Unzip the file
*   Upload unziped alpha-testimonials folder to your WordPress plugins directory (/wp-content/plugins/)
*   Activate from WordPress menu Plugins section
